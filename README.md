# IsentaDB

## Folder Structure

```yaml
IsentaDB
├─ src/
│   ├─ engine.rs
│   ├─ main.rs
│   ├─ parser.rs
│   ├─ storage.rs
│   └─ wal.rs
├─ target/
├─ .gitignore
├─ Cargo.lock
├─ Cargo.toml
├─ data.db
├─ data.wal
└─ README.md
```

## Build and Run

To build the project type this in your terminal

```bash
cargo build
```

To run the project type this in your terminal:

```bash
cargo run
```
